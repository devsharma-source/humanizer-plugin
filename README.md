# humanizer-plugin

A Claude Code plugin that removes AI writing patterns from text and makes it sound natural and human-written.

## Install

```bash
/plugin install <your-git-repo-url>
```

## Usage

```
/humanizer

[paste your text here]
```

## What it fixes

Based on Wikipedia's "Signs of AI writing" guide. Removes 24 documented AI patterns including em dash overuse, AI vocabulary words, promotional language, vague attributions, rule of three forcing, negative parallelisms, inline bold headers, emojis, and more.
